## 一、产品概述

### 1.1 产品定位

智能销售助手是一款面向**线上课程训练营销售团队**的AI驱动客户关系管理系统，通过整合多源数据、AI智能分析和自动化提醒，帮助销售人员在**4-5天训练营转化窗口**内高效管理客户、精准跟进、提升转化率。

### 1.2 业务背景

**业务模式**: 成人线上课程 - AI Agent副业培训
**业务流程**: 直播引流 → 4天试听训练营 → 销售跟进转化 → 购买正式课程
**团队规模**: 销售主管管理5-10名销售
**工作负载**:
- 每期新增学员: 120人/期
- 训练营频率: 每周一期（DAY0-DAY5）
- 长期培育池: 往期未成交学员累积增长，无清理机制

### 1.3 核心问题

| 痛点 | 现状描述 | 业务影响 |
|------|----------|----------|
| **120人/期记忆负担极重** | 每周新增120人，每人都有独特的痛点、顾虑、家庭情况，销售根本记不住 | 重复询问客户问题，体验差，转化率低 |
| **长期池无限累积失控** | 往期未成交学员持续累积（第5期已有570人），完全靠记忆不可能管理 | 高价值客户被埋没，激活困难 |
| **4-5天转化窗口压力大** | DAY0-DAY5必须完成转化，时间紧迫，需精准识别高意向学员 | 低效跟进导致转化率低 |
| **意向识别靠经验判断** | 凭感觉判断谁是高意向，容易遗漏S/A量学员 | 高价值客户流失，资源浪费 |
| **数据散落多系统** | 企微聊天、直播数据、试听课数据分散在3-4个系统 | 查询效率低，准备时间长 |

### 1.4 解决方案

**核心价值主张**: "让销售拥有超强记忆力，精准记住每个客户的所有细节，智能识别高意向学员，高效完成转化"

**四大核心能力**:
1. **智能记忆中心**: 客户360°视图，完整沟通历史，聊天记录智能检索
2. **AI意向分析**: 四维度评分（0-100分），自动分级（S/A/B），关键节点识别
3. **批次管理系统**: 本期营/往期池分层，长期池智能激活
4. **智能跟进助手**: AI话术推荐，优先级提醒，未成交原因分析

##### 功能3: AI意向分析引擎

**用户故事**: 作为销售，我希望系统自动识别客户的意向分级，避免遗漏高价值学员

**功能描述**:

**评分体系（统一且简单）**

```
【四维度评分】每个维度独立计算0-100分

维度1: 价格维度
├─ 分析因子: 价格反应、消费能力、付款意愿
└─ 评分规则:
    ├─ 90-100分: 主动询问付款方式，无价格异议
    ├─ 70-89分: 关注性价比，接受合理价格
    ├─ 50-69分: 价格敏感，希望优惠
    └─ <50分: 强烈要求打折，超出预算

维度2: 需求维度
├─ 分析因子: 痛点强度、紧迫性、主动性
└─ 评分规则:
    ├─ 90-100分: 痛点明确强烈，多次主动咨询
    ├─ 70-89分: 有需求但不紧迫
    ├─ 50-69分: 兴趣了解，痛点模糊
    └─ <50分: 被动了解，无明确需求

维度3: 共识维度
├─ 分析因子: 课程认可度、效果信心、竞品对比
└─ 评分规则:
    ├─ 90-100分: 全勤听课，高度认可
    ├─ 70-89分: 部分认可，有疑虑
    ├─ 50-69分: 观望态度，需要案例
    └─ <50分: 质疑效果，对比竞品

维度4: 信任维度
├─ 分析因子: 沟通深度、信息分享、互动质量
└─ 评分规则:
    ├─ 90-100分: 愿意深入沟通，主动分享
    ├─ 70-89分: 有所保留，需要背书
    ├─ 50-69分: 戒备心中等，观察态度
    └─ <50分: 戒备心强，质疑可靠性
```

```
【综合评分计算】
综合评分 = (价格维度 + 需求维度 + 共识维度 + 信任维度) / 4

【自动分级（根据综合评分）】
├─ S量: 综合评分 90-100分
├─ A量: 综合评分 80-89分
└─ B量: 综合评分 70-79分

【时间维度（后台维护，界面不显示）】
系统根据训练营日期自动判断学员所处阶段：
├─ DAY0之前: 训练营开始前阶段
├─ DAY0-DAY5: 训练营进行中阶段
└─ DAY5之后且未成交: 训练营结束后阶段

注意：时间维度仅用于后台逻辑判断和数据分析，
前端界面只显示S/A/B量分级标签，不显示时间阶段标签
```

**关键节点自动识别**

```
【规划卡片申请】
触发时机: DAY0-DAY1
识别方式:
├─ 客户回复"申请规划" 或 点击规划卡片链接
└─ 系统自动标记"已申请规划卡片✅"

【规划报价关单】
触发时机: DAY1-DAY5规划沟通过程中
识别方式:
├─ 销售在规划过程中推荐课程并报价
├─ 客户完成付款
└─ 系统自动标记"已成交💰"，状态变为"已成交"

【讨价还价识别】
触发时机: 规划报价后
识别方式:
├─ 客户回复包含"便宜点""打折""优惠"
├─ 或提出具体价格"4500能拿下吗"
└─ 提升综合评分，可能升级为S量

【确切付款时间】
触发时机: 规划报价后
识别方式:
├─ 客户回复"明天付款""等发工资""周五确定"
├─ 包含明确时间词
└─ 提升综合评分，可能升级为S量
```

**定时分析引擎**

```
【分析频率】
├─ 默认: 每1小时执行一次
├─ 可配置: 30分钟 / 1小时 / 2小时
└─ 智能触发: 仅在有新聊天记录或行为数据时执行

【分析流程】
1. 检测数据更新 (新聊天/新行为/新节点)
2. 提取特征 (LLM分析聊天内容)
3. 计算四维度评分 (价格/需求/共识/信任)
4. 计算综合评分 = 四维度平均分
5. 自动分级 (S/A/B)
6. 判断关键节点 (规划卡片/规划报价)
7. 生成AI分析建议
8. 更新客户评分和标签

【Buff手动调整】
销售可手动微调综合评分±5分:
├─ 场景: 客户特别积极/家庭支持/执行力强
├─ 操作: 点击"手动调整评分"按钮
├─ 范围: ±5分以内
└─ 记录: 调整原因和时间
```

**AI分析逻辑**

```
1. 数据收集
   ├─ 问卷填写内容
   ├─ 聊天记录（企微同步）
   ├─ 直播观看数据
   ├─ 试听课学习数据
   └─ 关键节点触发记录

2. 特征提取（LLM分析）
   ├─ 价格敏感度识别
   ├─ 需求痛点提取
   ├─ 课程认可度判断
   └─ 信任度评估

3. 四维度评分（各0-100分）
   ├─ 价格维度评分
   ├─ 需求维度评分
   ├─ 共识维度评分
   └─ 信任维度评分

4. 综合评分计算
   └─ 综合评分 = (价格+需求+共识+信任) / 4

5. 自动分级
   ├─ 90-100分 → S量
   ├─ 80-89分 → A量
   └─ 70-79分 → B量

6. 时间维度判断（后台维护）
   系统根据训练营日期自动判断学员所处阶段
   注意：此维度仅用于后台逻辑，不在界面显示

   └─ 针对性话术方向、注意事项
```
